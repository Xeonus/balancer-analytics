## Balancer V2 Analytics

An open sourced interface for Balancer V2 analytics - reimagined by Xeonus

Info URL: https://info.balancer.xeonus.io/#/

## Development

### Install Dependencies

```bash
yarn
```

### Run

```bash
yarn start
```

- Material-UI
